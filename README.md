AgriScan is a deep learning web application that detects crop diseases from leaf images in real time. Upload a photo of a plant leaf and get:
- ✅ **Disease name** and confidence score
- 💊 **Treatment recommendations** and prevention strategies
- 🔥 **GradCAM heatmap** showing what the model focused on
- 📊 **Top-5 predictions** with probability chart

## 🌿 Supported Crops & Diseases

AgriScan classifies **38 disease categories** across **14 crops**:

| Crop | Diseases Detected |
|------|------------------|
| Tomato | Early Blight, Late Blight, Bacterial Spot, Leaf Mold, Septoria Leaf Spot, Spider Mites, Target Spot, TYLCV, Mosaic Virus |
| Apple | Apple Scab, Black Rot, Cedar Apple Rust |
| Corn | Gray Leaf Spot, Common Rust, Northern Leaf Blight |
| Grape | Black Rot, Esca, Leaf Blight |
| Potato | Early Blight, Late Blight |
| Peach | Bacterial Spot |
| Pepper | Bacterial Spot |
| Orange | Citrus Greening (HLB) |
| Cherry | Powdery Mildew |
| Squash | Powdery Mildew |
| Strawberry | Leaf Scorch |
| Blueberry / Raspberry / Soybean | Healthy monitoring |

All crops also include a **Healthy** class.

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/agriscan.git
cd agriscan
```

### 2. Create a virtual environment
```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. (Optional) Train the model
Download the PlantVillage dataset and place it at `dataset/archive (1)/PlantVillage/PlantVillage/`:
```
dataset/
  plantvillage/
    Apple___Apple_scab/
    Apple___Black_rot/
    ...
```
Then run:
```bash
# Recommended for RTX 3060: Run via WSL2
wsl python3 train.py
```
This saves `model/agriscan_efficientnet.h5` and `model/class_names.json`.

> **Demo Mode**: If no trained model is found, the app automatically runs in **demo mode** with simulated predictions so you can explore the full UI.

### 5. Start the web app
```bash
python app.py
```
Open your browser at **http://127.0.0.1:5000**

## 🧠 Model Architecture

```
Input (224×224×3)
    ↓
EfficientNetB0 (ImageNet weights, frozen during warm-up)
    ↓
GlobalAveragePooling2D
    ↓
BatchNormalization
    ↓
Dense(512, ReLU) → Dropout(0.4)
    ↓
Dense(256, ReLU) → Dropout(0.3)
    ↓
Dense(38, Softmax)
```

**Two-Phase Training:**
1. **Warm-up (5 epochs, lr=1e-3)** — Only the classification head is trained; backbone is frozen.
2. **Fine-tuning (15 epochs, lr=1e-4)** — Top 30 layers of EfficientNetB0 are unfrozen and trained with a lower learning rate.

**Data Augmentation:** Rotation, shift, zoom, flip, brightness variation.

## 🔥 GradCAM Visualization

AgriScan uses **Gradient-weighted Class Activation Mapping (GradCAM)** to produce heatmaps that show which pixels in the leaf image contributed most to the predicted disease class. This makes the model's decisions interpretable and builds trust with users.

## 📈 Dataset

- **Source**: [PlantVillage Dataset](https://www.kaggle.com/datasets/emmarex/plantdisease)
- **Size**: ~87,000 labeled leaf images
- **Classes**: 38 (14 crops × multiple disease states + healthy)
- **Split**: 80% training / 20% validation (stratified)
